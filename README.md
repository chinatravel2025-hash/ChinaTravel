## 🌐 如何以网页形式查看

### 方法一：使用 GitHub Pages（推荐）

1. 在 GitHub 仓库中，点击 **Settings（设置）**
2. 在左侧菜单中找到 **Pages（页面）**
3. 在 **Source（源）** 部分，选择：
   - Branch: `main`（或你的主分支）
   - Folder: `/ (root)`
4. 点击 **Save（保存）**
5. 等待几分钟后，GitHub 会提供一个网址，格式如：`https://你的用户名.github.io/ChinaTravel/`
6. 访问该网址即可看到网页形式的文档

### 方法二：直接访问 HTML 文件

在 GitHub 仓库页面，点击 HTML 文件，然后点击右上角的 **"Raw"** 按钮旁边的 **"..."** 菜单，选择 **"View"** 或直接访问：

```
https://你的用户名.github.io/ChinaTravel/privacy_policy.html
```

### 方法三：本地预览

在本地克隆仓库后，直接在浏览器中打开 HTML 文件，或使用本地服务器：

```bash
# Python 3
python3 -m http.server 8000

# 或 Node.js
npx http-server
```

然后访问 `http://localhost:8000`
